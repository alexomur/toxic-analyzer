# AGENTS.md

## Текущий этап

Репозиторий находится на стадии `model-first` подготовки.

## Границы

- Не реализовывать заранее `frontend` и `backend`.
- Использовать ноутбуки только как исследовательскую лабораторию.
- Держать итоговый код только в обычных Python-модулях, а не в `.ipynb`.

## С чего начинать задачи по model

Для задач, связанных с `model/`, двигаться в таком порядке:

1. `model/AGENTS.md`
2. `model/README.md`
3. `model/pyproject.toml`
4. `model/src/`
5. `model/configs/` и `model/tests/`
6. `model/notebooks/` только если задача исследовательская

Если задача начинает расползаться в сторону преждевременной архитектуры продукта, нужно остановиться и вернуть объём работ в рамки `model/`.
